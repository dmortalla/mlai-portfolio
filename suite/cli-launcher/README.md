# MLAI CLI Launcher

Simple command-line launcher for your Machine Learning & AI Engineering
hero apps. It presents a text menu and calls `streamlit run` on the
selected app.

## Usage

From the root of your `mlai-portfolio` repo:

```bash
pip install -r suite/cli-launcher/requirements.txt
python suite/cli-launcher/launcher.py
```

Then choose an app number to launch.
